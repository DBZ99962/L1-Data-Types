# L1-Data-Types Setup Guide

## Quick Start

This repository contains comprehensive Python data types exercises for learning and practice.

## Prerequisites

- Python 3.7 or higher
- Jupyter Notebook
- pip (Python package installer)

## Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/DBZ99962/L1-Data-Types.git
cd L1-Data-Types
```

### Step 2: Create Virtual Environment (Optional but Recommended)

```bash
python -m venv venv

# On Windows:
venv\Scripts\activate

# On macOS/Linux:
source venv/bin/activate
```

### Step 3: Install Jupyter Notebook

```bash
pip install jupyter notebook
```

## Running the Notebook

### Start Jupyter Server

```bash
jupyter notebook
```

### Open the Notebook

1. Open your browser and navigate to `http://localhost:8888`
2. Navigate to `L1-Data-Types.ipynb`
3. Click to open the notebook

## Exercise Overview

The notebook contains 10 comprehensive exercises:

1. **Simple Interest Calculator** - Financial calculations
2. **Word Counter** - String operations
3. **List Statistics** - Sum, average, max, min operations
4. **Student Marks Average** - Nested list operations
5. **Coin Toss Game** - List manipulation and game logic
6. **Number Swapping** - Arithmetic operations without extra variables
7. **Alternative Indexing** - Complex list slicing
8. **Character Insertion** - String manipulation
9. **Dictionary Operations** - Dictionary creation and iteration
10. **Unique Words** - Set operations and ordered extraction

## Running Individual Cells

- Click on any cell
- Press `Shift + Enter` to run the cell
- Output appears below each cell

## Tips for Learning

1. Read through each exercise carefully
2. Try to solve before looking at the solution
3. Modify the solution to experiment
4. Test with different inputs
5. Add comments to understand each line

## Troubleshooting

### Jupyter not found
```bash
pip install --upgrade jupyter notebook
```

### Port 8888 already in use
```bash
jupyter notebook --port 8889
```

### Python not recognized
- Ensure Python is installed and added to PATH
- Use `python --version` to check
